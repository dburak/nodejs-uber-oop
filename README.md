# nodejs-uber-oop
First implementation of uber-like object-oriented web application.


## Technologies Used

**Client:** Vue 3 Composition API, Pinia

**Server:** NodeJS, Express, MongoDB

**Testing:** Jest

**Deployment:** Docker, Google Cloud Run
